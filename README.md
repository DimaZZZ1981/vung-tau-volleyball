# Vung Tau Beach Volleyball

Фан-сайт сообщества пляжного волейбола города Вунгтау, Вьетнам.

Публичная страница для игроков, новичков и спонсоров — без регистрации и лишнего.

## О проекте

Сайт создан для живого сообщества, которое собирается на пляже каждые выходные. Здесь можно узнать где и когда играем, посмотреть фото с игр, найти информацию о турнирах и связаться с организаторами.

## Что на сайте

- Расписание игр и тренировок
- Галерея фото и видео с игр
- Турниры — анонсы, результаты, фото
- Как попасть на первую игру
- FAQ для новичков
- Правила пляжного волейбола
- Информация о спонсорах
- Контакты организаторов (Telegram, Zalo)
